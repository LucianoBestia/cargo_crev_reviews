Trusted crates.io publishers

Every crate version on crates.io is published by a publisher.
It can be the author or an owner, or maybe not.
But this is the person that ultimately published the crate version and takes responsibility there is no malware inside.
Crates.io guarantees that once published, code can never be changed later.
The publisher is uniquely identified by the url of the git remote repository (Github, Gitlab or other).
